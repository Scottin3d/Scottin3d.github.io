Class Project - UI Canvas API for gEngine.

This UI Canvas API provides centralized support for the game designer to access UI controls such as buttons, sliders, toggles, ect. within the gEngine.


Class Project - Linux-style file system.

Using a pseudo operating system designed in Java, created a Linux-style file system. Through the use of the file system, user programs will now be able to access persistent data on disk through stream-oriented files instead of direct access to disk blocks with rawread() and rawrite().


Class Project - Behaviors, Bounds, and Interpolation.

In this programming assignment we defined simple behaviors demonstrating the understanding of extended bounding boxes, while working with controls governed by equations, and camera manipulations.


UWB Research - Leetcode Programming.

With a team of five, we are developing lessons that go beyond solving Leetcode programming problems that aim to help 3rd and 4th year UW Bothell CSSE student prepare for technical interviews.
